# 🌦️ Meteo App


A simple and interactive web application that lets you **check the current weather** in any city using the **OpenWeather API**.  
Stay informed about temperature, humidity, wind, sunrise/sunset, and enjoy dynamic backgrounds that change with the weather!

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6)  
- **API**: [OpenWeather API](https://openweathermap.org/api)  
- **Deployment**: [Vercel](https://vercel.com)  
- **Design**: Dynamic backgrounds, weather cards with shadows and hover effects  

---

## 🎯 Features

1. Search for a city by **name**  
2. Display:  
   - 🌡️ Current temperature  
   - 💧 Humidity  
   - 🌤️ Weather description  
   - 🌇 City name + country  
   - 🌅 Sunrise & sunset  
   - 🌬️ Wind speed  
3. Dynamic background depending on the weather (sunny, cloudy, rainy…)  
4. Responsive and modern interface
