# 🌦️ Meteo App

Une application web simple et interactive qui permet de **consulter la météo actuelle** d’une ville en utilisant l’API **OpenWeather**.

---

## **🚀 Démo en ligne**

Tu peux tester l’application ici : [https://meteo-app-chi.vercel.app](https://meteo-app-chi.vercel.app)

---

## **🛠️ Technologies utilisées**

- **Frontend** : HTML5, CSS3, JavaScript (ES6)  
- **API** : [OpenWeather API](https://openweathermap.org/api)  
- **Déploiement** : [Vercel](https://vercel.com)  
- **Design** : Fond dynamique, cartes météo avec ombre et effets hover  

---

## **🎯 Fonctionnalités**

1. Rechercher la météo d’une ville par son **nom**  
2. Afficher :  
   - 🌡️ Température actuelle  
   - 💧 Humidité  
   - 🌤️ Description du temps  
   - 🌇 Nom de la ville + pays  
   - 🌅 Lever et coucher du soleil  
   - 🌬️ Vitesse du vent  
3. Fond dynamique selon le temps (soleil, nuage, pluie…)  
4. Interface responsive et stylée

1. Cloner le dépôt :
```bash
git clone https://github.com/Yassine12c/meteo-app.git
Ouvrir index.html dans ton navigateur.

Option 2 : En ligne

Accéder directement à l’URL : https://meteo-app-chi.vercel.app

🔑 Configuration API

Insère ta clé OpenWeather dans script.js :

const apiKey = "TA_CLE_OPENWEATHER";


Pour sécuriser la clé sur Vercel : utilise Environment Variables (recommandé pour production).
